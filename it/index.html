Ciao e benvenuto su Homogram!

Homogram è un software on-premise leggero e semplice per condividere le tue immagini. E può essere utilizzato come archivio privato (screenshot, scatti privati, ecc.).

Homogram è rilasciato sotto licenza GPLv3, viene fornito AS-IS e non ci assumiamo alcuna responsabilità per il suo uso improprio.

Il nome *Homogram* deriva da uno scherzo sul nome 'instagram', infatti a differenza di quest'ultimo Homogram dà priorità agli homines. Tuttavia Homogram non nasce come sostituto ma proprio come il suo alter ego.

Primo passaggio, utilizzare la password del pannello laterale sinistro e i campi salt per creare l'hash da inserire nel file di configurazione. Ricordati di impostare manualmente lì anche il valore del sale.

Poiché eseguirai Homogram nel contesto del processo PHP, utilizzando un server Web limitato o un utente phpfpm, devi seguire alcune semplici direttive per una prima configurazione ottimale:
  1. Controlla i permessi di scrittura della tua cartella "HGRepo" nel percorso pubblico della tua app web; e imposta il suo percorso nel file di configurazione.
  2. Verifica i permessi di scrittura della tua cartella "dati" nel percorso privato della tua web app; e imposta il suo percorso nel file di configurazione.
  3. Imposta la lingua predefinita.
  4. Imposta il contesto predefinito su PUBLIC o PRIVATE (per un repository privato).
  5. Ogni cartella di immagini può contenere una sottocartella "thumbs" - che dovresti creare - per non perdere prestazioni: i nomi dei file nelle miniature devono essere uguali ai nomi dei file nelle cartelle principali.
